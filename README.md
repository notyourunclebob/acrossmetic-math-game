# Acrossmetic
A grid based random generated math game

rules: 
    -sums and operators are randomly chozen.
    -use numbers 1-9 to fill in the blanks to get the corret sums
    -all numbers count to both vertical and horizontal operations
    -order of operations is left to right and top to bottom (no BEDMAS)
    -only single digits can be used
    -all blanks must be filled
    -select options to change game grid size and operators used
    -number keys, arrow keys and mouse weel will change inputs
